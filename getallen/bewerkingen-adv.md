# Geavanceerde bewerkingen

Sommige geavanceerde bewerkingen kunnen worden gebruikt, waaronder:

* **Modulus**: ```x = y % 2```
* **Ophogen**: Gegeven a = 5
    * ```c = a++```, Resultaten: c = 5 and a = 6
    * ```c = ++a```, Resultaten: c = 6 and a = 6
* **Afnemen**: Gegeven a = 5
    * ```c = a--```, Resultaten: c = 5 and a = 4
    * ```c = --a```, Resultaten: c = 4 and a = 4
