# Bewerkingen

Je kunt wiskunde bewerkingen loslaten op getallen met wat simpele formules zoals:

* **Optellen**: ```c = a + b```
* **Aftrekken**: ```c = a - b```
* **Vermenigvuldigen**: ```c = a * b```
* **Delen**: ```c = a / b```

Je kunt haakjes gebruiken net zoals in wiskunde om formules op te delen: ```c = (a / b) + d```
